# QR Code Generator

A modern, offline-first QR Code Generator built using HTML, CSS, and JavaScript.

## Features
- Text & URL QR generation
- URL validation
- Multiple QR styles
- Color & background selection
- Optional logo upload
- Copy text & open link
- Download QR as PNG
- Works offline
- GitHub Pages compatible
